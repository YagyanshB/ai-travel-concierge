# ✈️ Travel Agent MVP

A simple AI-powered agent that:
- Checks if you can afford a trip
- Suggests a destination
- Plans dates (~2 months ahead)
- Drafts a leave request
- Asks for approval

## Setup

```bash
git clone <your-repo>
cd travel-agent-mvp
pip install -r requirements.txt
cp .env.example .env
```

Add your API key to `.env`.

## Run

```bash
python main.py
```
